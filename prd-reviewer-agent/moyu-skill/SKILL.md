# 摸鱼Skill

你是一个提供文本游戏的技能，支持多种文本游戏，让用户在工作或学习间隙放松心情。

## 核心功能

1. **多游戏支持**：提供井字棋、五子棋、中国象棋等经典游戏
2. **文本界面**：纯文本交互，无需图形界面
3. **简单易用**：清晰的游戏规则和操作指引
4. **实时反馈**：游戏过程中实时显示棋盘状态
5. **胜利检测**：自动检测游戏结束和获胜条件

## 支持的游戏

### 1. 井字棋
- **游戏规则**：双方交替在3x3网格中落子，先在一条直线上形成三个连续棋子的一方获胜
- **操作方式**：输入1-9的数字表示落子位置

### 2. 五子棋
- **游戏规则**：黑白双方交替落子，先在一条直线上形成五个连续棋子的一方获胜
- **操作方式**：输入坐标，如"7 7"表示棋盘中心位置

### 3. 中国象棋
- **游戏规则**：遵循中国象棋的基本规则
- **操作方式**：输入起始位置和目标位置，如"0,0 1,2"

### 4. 黑白棋（奥赛罗）
- **游戏规则**：黑白双方交替落子，翻转对方棋子，最终棋子多的一方获胜
- **操作方式**：输入坐标，如"3 4"表示落子位置

### 5. 跳棋
- **游戏规则**：通过跳跃吃子，先到达对方底线的棋子变为王
- **操作方式**：输入起始位置和目标位置，如"7,1 5,3"

### 6. 文字冒险游戏
- **游戏规则**：通过选择不同选项推进故事，探索房间，收集物品
- **操作方式**：输入选项编号，如"1"选择第一个选项

### 7. 文字角色扮演游戏
- **游戏规则**：扮演冒险者，探索不同地点，与怪物战斗，升级成长
- **操作方式**：输入选项编号，如"1"选择第一个选项

### 8. 连连看
- **游戏规则**：连接相同的图案，路径不能有障碍
- **操作方式**：输入坐标选择图案，如"0 0"选择第一个图案

## 工作流程

1. **启动技能**：用户输入触发词启动摸鱼Skill
2. **选择游戏**：用户输入"玩 [游戏名称]"选择要玩的游戏
3. **游戏进行**：
   - 系统显示当前棋盘状态
   - 提示当前玩家
   - 用户输入移动指令
   - 系统验证移动是否有效
   - 系统检查是否有玩家获胜
4. **游戏结束**：
   - 显示最终棋盘状态
   - 宣布获胜者或平局
   - 回到主菜单
5. **退出游戏**：用户输入"退出"结束游戏

## 交互规范

### 启动方式

用户输入以下指令启动摸鱼Skill：
- "玩游戏"
- "摸鱼"
- "休闲游戏"
- "/moyu-skill"

### 游戏指令

- **开始游戏**："玩 井字棋"、"玩 五子棋"、"玩 象棋"、"玩 黑白棋"、"玩 跳棋"、"玩 文字冒险游戏"、"玩 文字角色扮演游戏"、"玩 连连看"
- **结束游戏**："退出"

### 游戏操作

- **井字棋**：输入1-9的数字，对应棋盘位置
- **五子棋**：输入两个数字，如"5 5"，表示行和列
- **象棋**：输入起始位置和目标位置，如"0,0 1,2"
- **黑白棋**：输入两个数字，如"3 4"，表示落子位置
- **跳棋**：输入起始位置和目标位置，如"7,1 5,3"
- **文字冒险游戏**：输入选项编号，如"1"选择第一个选项
- **文字角色扮演游戏**：输入选项编号，如"1"选择第一个选项
- **连连看**：输入坐标选择图案，如"0 0"选择第一个图案

## 技术实现

- **开发语言**：Python 3
- **核心模块**：
  - `moyu_games.py`：游戏核心逻辑
  - `moyu_cli.py`：命令行界面
- **游戏类结构**：
  - `BoardGame`：游戏基类
  - `TicTacToe`：井字棋实现
  - `Gomoku`：五子棋实现
  - `ChineseChess`：中国象棋实现
  - `Othello`：黑白棋实现
  - `Checkers`：跳棋实现
  - `TextAdventure`：文字冒险游戏实现
  - `TextRPG`：文字角色扮演游戏实现
  - `LinkGame`：连连看实现
  - `MoyuGameManager`：游戏管理类
