# 摸鱼Skill (moyu-skill)

## 项目简介

摸鱼Skill 是一个提供多种文本游戏的技能，让用户在工作或学习间隙放松心情。通过纯文本交互的方式，用户可以随时随地享受经典游戏的乐趣，无需图形界面。

## 功能特性

- **多游戏支持**：提供井字棋、五子棋、中国象棋、黑白棋、跳棋、文字冒险游戏、文字角色扮演游戏和连连看等多种经典游戏
- **文本界面**：纯文本交互，无需图形界面，可在任何终端环境中运行
- **简单易用**：清晰的游戏规则和操作指引，适合各个年龄段的用户
- **实时反馈**：游戏过程中实时显示棋盘状态，提供即时反馈
- **胜利检测**：自动检测游戏结束和获胜条件，确保游戏的公平性

## 支持的游戏

### 1. 井字棋
- **游戏规则**：双方交替在3x3网格中落子，先在一条直线上形成三个连续棋子的一方获胜
- **操作方式**：输入1-9的数字表示落子位置

### 2. 五子棋
- **游戏规则**：黑白双方交替落子，先在一条直线上形成五个连续棋子的一方获胜
- **操作方式**：输入坐标，如"7 7"表示棋盘中心位置

### 3. 中国象棋
- **游戏规则**：遵循中国象棋的基本规则
- **操作方式**：输入起始位置和目标位置，如"0,0 1,2"

### 4. 黑白棋（奥赛罗）
- **游戏规则**：黑白双方交替落子，翻转对方棋子，最终棋子多的一方获胜
- **操作方式**：输入坐标，如"3 4"表示落子位置

### 5. 跳棋
- **游戏规则**：通过跳跃吃子，先到达对方底线的棋子变为王
- **操作方式**：输入起始位置和目标位置，如"7,1 5,3"

### 6. 文字冒险游戏
- **游戏规则**：通过选择不同选项推进故事，探索房间，收集物品
- **操作方式**：输入选项编号，如"1"选择第一个选项

### 7. 文字角色扮演游戏
- **游戏规则**：扮演冒险者，探索不同地点，与怪物战斗，升级成长
- **操作方式**：输入选项编号，如"1"选择第一个选项

### 8. 连连看
- **游戏规则**：连接相同的图案，路径不能有障碍
- **操作方式**：输入坐标选择图案，如"0 0"选择第一个图案

## 安装和使用

### 环境要求
- Python 3.x

### 安装步骤
1. 克隆或下载本项目到本地
2. 进入项目目录

### 使用方法
1. 运行游戏管理器：
   ```bash
   python3 scripts/moyu_cli.py
   ```
2. 按照提示选择要玩的游戏
3. 按照游戏内提示进行操作
4. 输入"退出"结束游戏

## 技术实现

- **开发语言**：Python 3
- **核心模块**：
  - `moyu_games.py`：游戏核心逻辑，包含所有游戏的实现
  - `moyu_cli.py`：命令行界面，处理用户输入和游戏启动
- **游戏类结构**：
  - `BoardGame`：游戏基类，定义通用游戏接口
  - 各种具体游戏类：继承自 `BoardGame`，实现各自的游戏逻辑
  - `MoyuGameManager`：游戏管理类，负责游戏的启动和运行

## 游戏操作说明

### 启动方式
用户输入以下指令启动摸鱼Skill：
- "玩游戏"
- "摸鱼"
- "休闲游戏"
- "/moyu-skill"

### 游戏指令
- **开始游戏**："玩 井字棋"、"玩 五子棋"、"玩 象棋"、"玩 黑白棋"、"玩 跳棋"、"玩 文字冒险游戏"、"玩 文字角色扮演游戏"、"玩 连连看"
- **结束游戏**："退出"

### 游戏操作
- **井字棋**：输入1-9的数字，对应棋盘位置
- **五子棋**：输入两个数字，如"5 5"，表示行和列
- **象棋**：输入起始位置和目标位置，如"0,0 1,2"
- **黑白棋**：输入两个数字，如"3 4"，表示落子位置
- **跳棋**：输入起始位置和目标位置，如"7,1 5,3"
- **文字冒险游戏**：输入选项编号，如"1"选择第一个选项
- **文字角色扮演游戏**：输入选项编号，如"1"选择第一个选项
- **连连看**：输入坐标选择图案，如"0 0"选择第一个图案
